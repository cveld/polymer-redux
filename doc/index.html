Polymer - Redux

Some time ago I realised that the two best things currently out there, Polymer and Redux, were not yet used together. So I decided to dig into this and prove that they work great together! I mist admit that React and Angular are great frameworks too, but why invent a whole new concepts, while the DOM itself has a lot of framework type concepts just built into it:

  1. Component model (custom element)
  2. Concept of data flow (attributes/properties and events)
  3. declarative syntax (HTML)
Anyway, this is not a post about Polymer :)

Below I'll explain how I use a single store, no two-way data-binding, actions and reducers!
